Virtual6502

Virtual6502 is an emulator for the NES' 6502 CPU, written in Python. It provides two modes - interactive and from-file. The interactive prompt allows you to write 6502 Assembly straight from an interactive command-prompt. It also provides special functions to display the state of the CPU. Loading a file runs all of the 6502 Assembly instructions defined in the file.
